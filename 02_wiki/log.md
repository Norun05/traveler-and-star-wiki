# 여행자와 별 위키 로그

## 문서 목적

본 문서는 여행자와 별 위키의 작업 로그 문서이다.
시간순으로 어떤 문서를 만들었는지, 어떤 기준이 갱신되었는지, 무엇이 아직 보류 상태인지 기록한다.

이 문서는 설정 원문이 아니라 위키 운영 기록을 남기는 문서이다.

### 현재 기획 정리 문서 생성

현재 살아 있는 기획 기준을 상위 문서로 정리하였다.
주요 반영 항목은 다음과 같다.

- 주인공은 IEB 기록관으로서 행성에 파견된다.
- 이동수단은 워프 스테이션의 워프 열차이다.
- 파트너 동행은 제거 방향을 우선 검토한다.
- 탐색은 선택지 기반 구조를 우선한다.
- 센터는 퀘스트, 보고, 지원, 목표 갱신의 허브이다.
- 생존 스탯은 체력, 기력, 정신, 허기 4종을 사용한다.
- 기초 스탯은 근력, 섬세함, 지능을 기본 축으로 둔다.
- 하우징은 존재하지만 현재 개발 우선순위는 낮다.
- 전투는 주 콘텐츠가 아니라 위협 대응 성격으로만 검토 중이다.
- 모든 행성의 달력은 1년 4달, 1달 28일의 공통 규칙을 따른다.

### 보류 상태로 함께 기록한 항목

현재 기획 정리 문서에는 아직 확정되지 않은 항목도 별도 섹션으로 기록하였다.

주요 보류 항목 예시:
- 파트너 최종 제거 여부
- 정기 보고의 주기와 구조
- 주간 우선 분야 선택 방식
- 랜덤 조우의 세부 규칙
- 전투 또는 위협 대응의 실제 도입 범위
- 지원품 상향 기준
- 호감도 및 연애 세부 규칙
- 기본 스탯 가시화 방식

## 현재 기준 요약

현재 위키 운영 기준은 다음과 같다.

- 살아 있는 기획은 `현재_기획_정리.md` 를 상위 기준으로 삼는다.
- 인덱스 문서는 문서 위치와 상태를 안내한다.
- 로그 문서는 위키의 변화 과정을 기록한다.
- 개별 설정 문서는 현재 기획 정리 문서와 충돌할 경우 재검토 대상이 된다.
